# SPI Master Design using Verilog HDL

> RTL implementation of an SPI (Serial Peripheral Interface) Master with FSM-based control logic, developed in Verilog HDL and verified through simulation.

---

## Project Overview

This project presents the design and functional verification of an **SPI Master** compliant with the standard four-wire SPI interface. The controller is implemented using **Verilog HDL** with a Finite State Machine (FSM) to manage serial communication between the master and slave devices.

The objective of this project is to strengthen RTL design skills by implementing a widely used serial communication protocol while following good hardware design practices.

---

## Key Features

* RTL implementation in Verilog HDL
* FSM-based control architecture
* 8-bit serial data transmission
* Master-controlled serial clock generation
* MOSI and MISO data handling
* Active-Low Chip Select (CS)
* Functional verification using a dedicated testbench
* Synthesizable RTL

---

## SPI Interface

| Signal | Direction      | Description                           |
| ------ | -------------- | ------------------------------------- |
| MOSI   | Master → Slave | Serial data transmitted by the master |
| MISO   | Slave → Master | Serial data received from the slave   |
| SCLK   | Master → Slave | Serial clock generated by the master  |
| CS     | Master → Slave | Active-Low Slave Select               |

---



## Finite State Machine

| State    | Description                                |
| -------- | ------------------------------------------ |
| IDLE     | Waits for the start signal                 |
| TRANSFER | Performs serial transmission and reception |
| FINISH   | Completes communication and asserts `done` |
